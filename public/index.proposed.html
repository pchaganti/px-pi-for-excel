AI help inside Excel, in plain English.

Pi for Excel is a friendly sidebar assistant for spreadsheets. Ask normal questions, get clear answers, and automate repetitive work without leaving your workbook.

  • Works with your model of choice (Claude, GPT, Gemini, and more).
  • No coding needed to install — download one file and add it in Excel.
  • Your workbook stays in Excel; you stay in control of what gets changed.

What this is (and how it feels to use)

Built for non-technical users who still want serious spreadsheet help.

1) Ask naturally “Summarize this selection”, “fix this formula”, or “find where EBITDA is used”.
2) Keep context Pi can read your current sheet/selection so you spend less time explaining what you’re looking at.
3) Stay in control You can review responses, interrupt with Esc, and decide when to apply edits.

Install + get started (about 2 minutes)

No dev tools required. If you can upload a file in Excel, you can install this.

1

Download manifest.prod.xml using the green button above.

2

Add it in Excel:

  • Windows: Insert → My Add-ins → Upload My Add-in
  • macOS: copy the file to Excel’s wef folder (full guide linked above)
3

Open Pi from the ribbon and run /login once.

4

Send your first prompt, for example:

  • “What sheet am I on?”
  • “Summarize my current selection.”
  • “Find formulas referencing Inputs.”
Tip: API key login is usually the easiest path. OAuth can need a local HTTPS proxy helper on some Excel/macOS setups.
