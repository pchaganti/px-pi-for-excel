Pi for Excel

An open-source, multi-model AI sidebar add-in for Microsoft Excel — powered by Pi. Bring your own key. Your workbook stays local.

BYOK Claude • GPT • Gemini • … Sessions Excel tools Extensions

Install (2 minutes)

  1. Download manifest.prod.xml.
  2. In Excel: Insert → Add-ins → My Add-ins → Add from file / Upload.
  3. Open Pi from the ribbon, then run /login.
Updates are automatic: the manifest points to this hosted URL. When we deploy a new build, you usually get it next time you open the taskpane.

Get the files

Download manifest.prod.xml View source on GitHub Open taskpane.html (diagnostics)

Note: some OAuth/subscription flows may require a local HTTPS proxy helper on macOS Excel (WKWebView mixed-content restrictions).

What can it do?

  1. Read your selection automatically
  2. Write formulas + format ranges
  3. Search workbook + trace dependencies
  4. Persist sessions (resume later)
  5. Extend with slash-command extensions
Tip: type / to browse commands. Press Esc to interrupt.
